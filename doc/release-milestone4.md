# Milestone 4 Release Steps

1. Ensure `main` contains final Milestone 4 code and docs.
2. On GitHub, open the repository Releases page.
3. Click **Draft a new release**.
4. Tag/version:
   - Tag: `milestone-4`
   - Release title: `milestone-4`
5. In release notes, summarize:
   - final layout and usability improvements
   - feedback-driven chart updates
   - deployment-ready state
6. Publish release before the submission deadline.
7. Copy the release URL into `canvas-submission.md`.
8. Submit that release URL to Canvas.
